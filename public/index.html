🔗 Abrir Feed no Navegador

Como configurar no ImovelWeb CRM

  1. Acesse o painel do ImovelWeb CRM.
  2. Vá em Configurações → Integração de Feed.
  3. Cole a URL do seu feed acima.
  4. Selecione o formato XML Customizado.
  5. Salve. O ImovelWeb irá buscar os dados automaticamente.
